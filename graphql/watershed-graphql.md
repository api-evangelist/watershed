# Watershed GraphQL API

Watershed is an enterprise climate platform for measuring, reducing, and reporting carbon emissions. The API covers footprint data ingestion, emission factor lookups, Scope 1/2/3 calculations, supplier data requests, and climate reporting (GHG Protocol, CDP, TCFD).

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Watershed REST API
**Documentation:** https://watershed.com/platform

- Reference: https://watershed.com/platform
- Schema: https://github.com/api-evangelist/watershed
